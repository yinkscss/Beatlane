Web Audio runtime

G4

Locked audio architecture — no Howler/Tone.

  • Shared AudioContext; master → music + SFX gains; mute via gain.
  • One-shot SFX buffers; one sample bed track (local or Storage).
  • Run start syncs music start time; mute toggle in UI.
Done when: track + SFX play; mute works; no extra audio libs.
Refs: PRD §5.3 · STACK audio note

Chart engine

G5

JSON charts drive tiles; chart is source of truth.

  • Chart schema: BPM/offset, notes (lane, time, type), events (SPEED UP).
  • ≥2 sample charts (Easy/Normal); loader from public or Storage.
  • SPEED UP banner + countdown ring; scroll rate jump.
  • Tiles scheduled from chart + music clock (not waveform analysis).
Done when: both sample charts play through; SPEED UP fires mid-run.
Refs: PRD §5.1–5.3
